Onchange

El evento onchange se produce cuando el valor de un elemento se ha cambiado. Para botones de radio y casillas de verificación, el evento onchange ocurre cuando el estado de activación se ha cambiado.

OnKeyDown

El evento onkeydown se produce cuando el usuario presiona una tecla (en el teclado).

🔝

OnKeyPress y OnKeyUp

El evento onkeypress se produce cuando el usuario presiona una tecla (en el teclado)

Nota:El evento onkeypress no se activa para todas las teclas (por ejemplo, ALT, CTRL, SHIFT, ESC) en todos los navegadores. Para detectar solo si el usuario ha presionado una tecla, use el evento onkeydown en su lugar, ya que funciona para todas las teclas.

OnKeyUp

El evento onkeyup se produce cuando el usuario suelta una tecla (en el teclado).

🔝

Onload

El evento onload se produce cuando se ha cargado un objeto.

OnMouseDown And OnMouseUp

El evento onmousedown se produce cuando un usuario presiona un botón del mouse sobre un elemento.

El evento onmouseup se produce cuando un usuario suelta un botón del ratón sobre un elemento.

¡Haz clic en el texto! La función mouseDown() se activa cuando se presiona el botón del mouse sobre este párrafo y establece el color del texto en gris. La función mouseUp() se activa cuando se suelta el botón del mouse y establece el color del texto en verde.

OneMouseMove - OneMouseOut and OnMouseOver

onmousemove:
Mouse over me!

onmouseout:
Mouse over me!

onmouseover:
Mouse over me!

Este ejemplo demuestra la diferencia entre onemousemove, onemouseout y onemouseover.

El evento onmousemove se produce cada vez que el puntero del mouse e mueve sobre el elemento div.

El evento onmouseout se produce cuando el puntero del mouse se mueve fuera del elemento div y cuando deja sus elementos secundarios (p y span).

El evento onmouseover se produce cuando el puntero del mouse entra en el elemento div y sus elementos secundarios (p y span).
